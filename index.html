Proyectos académicos y profesionales en los que he participado, enfocados en desarrollo de software y tecnología aplicada.

Proyecto 1 — Sistema de entrenamiento comercial con IA

2024 · Next.js / ASP.NET Core / Inteligencia Artificial

Desarrollo de un sistema interactivo de capacitación comercial basado en un avatar con inteligencia artificial. El sistema simula conversaciones de venta y proporciona retroalimentación para mejorar habilidades comerciales. Proyecto desarrollado durante mi estadía profesional.

Proyecto 2 — Backend para plataforma de capacitación interna

2024 · ASP.NET Core / APIs REST

2024 · Python / Procesamiento de lenguaje natural

Diseño y desarrollo de una landing page personal con enfoque visual moderno, estructura semántica clara y base técnica para su evolución a Progressive Web App (PWA).
